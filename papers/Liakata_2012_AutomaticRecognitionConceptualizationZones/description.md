Defines the 11-label Core Scientific Concepts discourse scheme for full scientific articles and evaluates SVM, LibLinear, and CRF sentence classifiers on 265 chemistry and biochemistry papers, making it directly useful for redesigning repository note structure around hypothesis, method, experiment, observation, result, and conclusion roles.
