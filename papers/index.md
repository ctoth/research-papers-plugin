## Jonnalagadda_2015_AutomatingDataExtractionSystematic  (systematic-reviews, information-extraction, biomedical-nlp, data-extraction, literature-review)
Systematically reviews pre-2015 NLP and rule-based methods for extracting systematic-review data elements from abstracts and full text, cataloging 52 candidate fields, 26 included studies, and the main gap: no unified framework that covers the full review data abstraction workflow.

## Liakata_2012_AutomaticRecognitionConceptualizationZones  (scientific-discourse, sentence-classification, information-extraction, core-scientific-concepts, conditional-random-fields)
Defines the 11-label Core Scientific Concepts discourse scheme for full scientific articles and evaluates SVM, LibLinear, and CRF sentence classifiers on 265 chemistry and biochemistry papers, making it directly useful for redesigning repository note structure around hypothesis, method, experiment, observation, result, and conclusion roles.

## Schulz_2010_CONSORT2010StatementUpdated  (reporting-guidelines, randomized-trials, clinical-trials, checklists, evidence-synthesis)
Defines the CONSORT 2010 reporting checklist and flow diagram for parallel-group randomized trials, making it useful as a structured extraction target or completeness rubric for clinical-trial papers rather than as an algorithmic NLP paper.

## Teufel_1999_AnnotationSchemeDiscourse-levelArgumentation  (annotation, argumentation, discourse-analysis, scientific-writing, summarization)
Introduces a sentence-level annotation scheme for scientific papers that labels local argumentative role rather than global rhetorical structure, then validates the scheme with inter-annotator studies. Highly relevant as an early precursor to argument zoning and for any project that needs reproducible discourse labels for research articles.

## Elm_2007_StrengtheningReportingObservationalStudies  (reporting-guidelines, checklists, evidence-synthesis, observational-studies, information-extraction)
Introduces the 22-item STROBE reporting checklist for cohort, case-control, and cross-sectional studies, making it useful as a completeness rubric or extraction schema for observational-study papers rather than as an algorithmic methods paper.

## Marshall_2017_AutomatingBiomedicalEvidenceSynthesis  (systematic-reviews, biomedical-nlp, evidence-synthesis, clinical-trials, information-extraction)
Presents RobotReviewer, an open-source web and API system that processes full-text randomized controlled trial PDFs and generates structured evidence-synthesis reports with risk-of-bias and PICO-related outputs. Its main technical contributions are rationale-linked risk-of-bias models, supervised-distant-supervision for full-text PICO sentence extraction, and aspect-specific PICO embeddings for retrieval and visualization. It is highly relevant to this project as a concrete prior architecture for traceable extraction over biomedical trial papers rather than generic article summarization.
