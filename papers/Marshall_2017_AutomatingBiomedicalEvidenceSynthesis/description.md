Presents RobotReviewer, an open-source web and API system that processes full-text randomized controlled trial PDFs and generates structured evidence-synthesis reports with risk-of-bias and PICO-related outputs.
Its main technical contributions are rationale-linked risk-of-bias models, supervised-distant-supervision for full-text PICO sentence extraction, and aspect-specific PICO embeddings for retrieval and visualization.
It is highly relevant to this project as a concrete prior architecture for traceable extraction over biomedical trial papers rather than generic article summarization.
